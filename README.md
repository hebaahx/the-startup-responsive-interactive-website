In dit project is het de bedoeling dat ik een responsive webite opbouw, namenlijk een weather app. Deze site is zowel bruikbaar voor telefoon, ipad en desktop. Met voor telefoon en desktop een andere layout.
De volgende onderdelen zijn hierbij toegepast:
* Responsive
  De website is responsive opgebouwd. Met als desgin dat alle content onder elkaar staat in een one column layout en voor op desktop dat de content met gebruik van grid        verdeeld is over het scherm. Voor mobile is gekozen voor een logische volgorde, van een intro over de app tot aan activiteiten. Deze volgorde is belangrijk want zo komt   
  belangrijke content als eerst te zien voor de gebruiker. Voor de layout op desktop is er meer ruimte en van daar ook dat de content meer verdeeld met aan de linker kant de 
  intro en het weer voor de komende vijf dagen en een input met dat je je kan aanmelden voor de daily forecast. Aan de rechterkant is het huidige weer en de activiteiten die 
  je het beste wel en niet kan doen met dit weer.
* Toegankelijkheid
  Verder heb ik rekening gehouden met de toegankelijkheid van de page. Het is abelangrijk dat de page voor zo veel mogelijk mensen beruikbaar zit. Dit heb ik gedaan door 
  middel van WCAG-audit testen uit te voeren
* Huisstijl
  Voor de opdrachtgever was het belangrijk dat ik me houd aan de huisstijl van DEPT. Dit houdt in dat ik de kleuren en typografie van DEPT gebruik. ik heb dit uitgevoerd met 
  behulp van een stylehguide. Eerst heb ik een interface inventory opgebouwd waarin ik alles heb verzamelt wat het zelfde is of vaak voorkomt. Hierdoor heb ik meer inzicht 
  gekregen en was het makkelijker om een styleguide te maken en de huisstyle toe te passen. 
* Interactie
  De page heeft ook een interactie. Er is een stuk dat je je kan aanmelden voor de daily forecast. Het leek me leuk en practisch voor de opdrachtgever als de button een        animatie heeft en daar in heb ik gelijk 




Dit is mijn live link op github: https://hebaahx.github.io/the-startup-responsive-interactive-website/
